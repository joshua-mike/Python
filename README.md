# Python ML Data Tools

A collection of Python utilities for handling and preparing data for machine learning workflows.

## Tools

### Scraper

Web scraping and file download utilities for collecting ML training data.

- **pyscraper.py** - Scrapes structured data (e.g., tables) from web pages and exports to CSV/DataFrame using BeautifulSoup and pandas.
- **pydownload.py** - Bulk file downloader with TLS compatibility handling, designed for grabbing datasets from web sources.

## Requirements

- Python 3.x
- requests
- beautifulsoup4
- pandas
